# E2E Tests

Тесты для проверки фронтенда с использованием Playwright.

## Запуск тестов

```bash
# Запустить все тесты
npm run test:e2e

# Запустить с UI (интерактивный режим)
npm run test:e2e:ui

# Запустить с видимым браузером
npm run test:e2e:headed
```

## Структура тестов

```
tests/
├── basic.spec.ts         # Базовые тесты страниц
└── frontend-check.spec.ts # Проверка на ошибки в консоли
```

## Что проверяют тесты

1. **basic.spec.ts**
   - Страница логина загружается корректно
   - Есть поля email и password
   - Главная страница показывает загрузку или редиректит

2. **frontend-check.spec.ts**
   - Страница логина
   - Главная страница без критических ошибок в консоли
   - Скриншоты сохраняются при падении

## Скриншоты

При запуске тестов скриншоты сохраняются:
- `login-page.png` - страница логина
- `main-page.png` - главная страница
- При ошибках: `before-login.png`

## CI/CD

В CI автоматически:
- Ретраится при неудаче (до 2 раз)
- Сохраняются скриншоты и видео при падении
- Отключается `reuseExistingServer`
