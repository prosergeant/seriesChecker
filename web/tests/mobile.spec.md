# tests/mobile.spec.ts

Playwright E2E тесты мобильной раскладки при viewport 375×812 (iPhone).

## Что тестируется
- Страница `/login` не имеет горизонтального overflow (`scrollWidth ≤ 375`)
- Email-инпут шире 200px (не обрезан)
- Главная страница без авторизации редиректит на `/login`
- Главная страница с авторизацией — нет горизонтального overflow (требует `TEST_EMAIL`/`TEST_PASSWORD`)
- Фильтры имеют `overflow-x-auto` класс (горизонтальный скролл, а не перенос)

## Переменные окружения
- `TEST_EMAIL`, `TEST_PASSWORD` — для тестов на авторизованных страницах; без них тесты пропускаются

## Что изменено и почему
Создан с нуля. Дополняет PR #6 (mobile responsive) автоматизированными проверками layout на мобильном viewport.
