# PlayerModal.tsx

## Что делает
Кнопка "Смотреть" + модальное окно с iframe плеером для фильма/сериала.

Используется в `ProgressCard` (page.tsx) вместо прежней кнопки, открывавшей sspoisk.ru в новой вкладке.

## Логика
- При открытии рендерит `<iframe src="https://fbfree.lat/{series|film}/{kinopoisk_id}">`
- Браузер пользователя загружает страницу, Kinobox JS запускается с его IP → токены плеера валидны
- iframe рендерится только когда модалка открыта (`isOpen && <iframe ...>`) чтобы не делать запрос заранее

## Почему не через resolver
Resolver (`/api/series/{id}/resolve`) запускает Chromium на сервере — токены привязываются к IP сервера, а не пользователя → 404 при переходе. Клиентский iframe решает это автоматически.
